**一套面向外企工作场景的 AI 邮件处理工具集**，包含中文邮件润色和双向翻译（中↔英）两大核心功能，帮助你把口语化的草稿快速转化为结构清晰、措辞专业的外企商务邮件。

---

## ✨ 功能特点

### 1. 邮件润色器（Python 版）
- **输入**：口语化的中文草稿（如会议记录、需求描述）
- **输出**：格式完整的中文外企邮件（含主题、称呼、正文、结尾、署名）
- **适用场景**：日常邮件起草、周报整理、需求沟通

### 2. 双向邮件翻译器（Scriptable 版）
- **输入**：中文或英文邮件草稿
- **输出**：对应语言的完整外企邮件（自动检测语言方向）
- **适用场景**：与外籍同事沟通、英文邮件起草、中英双语团队协作

| 工具 | 语言/平台 | 依赖 |
